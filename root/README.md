# Enable root access


On the printers touchscreen click "Settings" and scroll down and select "Root account information"

<img src=https://github.com/DnG-Crafts/K2-RFID/blob/main/root/1.jpg"><br>

<img src=https://github.com/DnG-Crafts/K2-RFID/blob/main/root/2.jpg"><br>

Check "I have read and understood the risks of Root login" and wait 30 seconds then press "Ok"

<img src=https://github.com/DnG-Crafts/K2-RFID/blob/main/root/3.jpg"><br>

You will then be shown the root username and password and root access is enabled

<img src=https://github.com/DnG-Crafts/K2-RFID/blob/main/root/4.jpg"><br>
