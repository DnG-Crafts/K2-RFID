# Enable root access


### On the printers touchscreen click "Settings" and scroll down and select "Root account information"

<img src="https://github.com/DnG-Crafts/K2-RFID/blob/main/root/1.jpg" width="50%" height="50%"><br>

<img src="https://github.com/DnG-Crafts/K2-RFID/blob/main/root/2.jpg" width="50%" height="50%"><br><br><br>

### Check "I have read and understood the risks of Root login" and wait 30 seconds then press "Ok"

<img src="https://github.com/DnG-Crafts/K2-RFID/blob/main/root/3.jpg" width="50%" height="50%"><br><br><br>

### You will then be shown the root username and password and root access is enabled
